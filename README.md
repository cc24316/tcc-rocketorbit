# RocketOrbit 🚀🌌🐱


Aplicativo de produtividade voltado para auxiliar no foco durante atividades, combinando monitoramento de uso do dispositivo com gamificação para incentivar a consistência e o foco e reduzir distrações.


## Objetivo:

Ajudar o usuário a iniciar e manter o foco por meio de acompanhamento de tempo, metas personalizáveis e feedback visual.


## Funcionalidades:

* Monitoramento automático de tempo (foco x uso do celular)
* Sistema de missões com progresso em órbita (foguete -> planeta)
* Definição de metas personalizáveis para estudo, trabalho ou hobbies
* Controle de tempo de uso do celular com limite diário personalizável
* Sistema de recompensas baseado em consistência
* Pausas inteligentes durante o uso


## Gamificação:

* Sessões de foco representam progresso em direção a um destino (planeta)
* O usuário “entra em órbita” ao iniciar uma atividade, tarefa ou hobby
* O progresso é mantido enquanto há foco ativo
* Uso de aplicativos distrativos interrompe o avanço
* Cumprir metas diárias gera recompensas acumulativas


## Tecnologias:

Android Studio
Kotlin
Monitoramento de uso de apps (UsageStatsManager)
